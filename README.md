# Playgrounds

This is my personal training repository for iOS/Swift development. It contains structured coding challenges across algorithm topics and Swift language fundamentals, used to build interview readiness and sharpen core skills over time.

## Structure

Each challenge lives in a named Swift playground filed under its topic folder. Challenges follow a consistent format — a short story sets the scene, pre-defined test data is provided, and the solution section is left empty to fill in.

### Algorithm Topics
- **BinarySearch/** — binary search patterns
- **Hashing/** — dictionary and set-based problems
- **HigherOrderFunctions/** — map, filter, reduce
- **LinkedList/** — linked list traversal and manipulation
- **MatrixTraversal/** — 2D array problems
- **SlidingWindow/** — fixed and variable window techniques
- **Stacks/** — stack-based problems
- **TwoPointers/** — converging, slow/fast, and multi-array pointer patterns

### Swift Development
- **SwiftDev/Concurrency/** — async/await, actors, race conditions
- **SwiftDev/Fundamentals/** — closures, generics, protocols, enums
- **SwiftDev/Networking/** — Codable, URLSession, API architecture

## Difficulty Scale
Challenges are rated 1–10 and included in the filename — e.g. `TheBeacon(2)`, `OrbitalCommand(7)`.
